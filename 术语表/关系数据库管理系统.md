# 关系数据库管理系统

关系数据库管理系统（英语：Relational Database Management System，缩写为 RDBMS）是管理关系数据库的数据库管理系统。

关系数据库是创建在关系模型基础上的数据库，借助于集合代数等数学概念和方法来处理数据库中的数据。关系数据库将数据间的关系以数据库表的形式加以表达，并将数据存储在表格中，以便于查询。
