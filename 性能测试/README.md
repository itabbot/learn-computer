# 性能测试<!-- omit in toc -->

软件性能测试用于评估和验证系统的性能、容量、稳定性等质量属性。通常有以下应用场景：

- 新系统上线：探知系统能力，防止一上线即被用户流量打垮。
- 技术升级验证：大的技术架构升级后进行性能评估，验证新技术场景的系统性能状态。
- 峰值稳定性：大促活动等峰值业务稳定性考验，保障峰值业务不受损。
- 容量规划：对系统进行精细化的容量规划，分布式系统机器资源分配。
- 性能瓶颈探测：探测系统中的性能瓶颈点，以进行针对性优化。
