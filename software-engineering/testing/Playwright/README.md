### 1.2. 定义

Playwright 是一个由 Microsoft 开发的开源自动化测试工具，是专门为满足端到端测试的需求而创建的。它支持多种浏览器渲染引擎，包括 Chromium、WebKit 和 Firefox。Playwright 支持在 Windows、Linux 或 macOS 等操作系统上进行本地或持续集成（CI）测试，可以选择在无头模式（headless，无可见浏览器界面）下进行，也可以选择在有界面模式下进行，并且可以进行原生移动设备仿真测试。
